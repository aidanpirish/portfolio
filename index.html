Westminster Learning Goals

Westminster College has college wide learning goals that all computer science students must demonstrate for graduating. Listed below is a list of those goals with an example of how I demonstrated that goal in a project. More information on the projects can be found in the "projects" tab.

  • To attain a system level understanding of the computer.

    Onescriptsite is a bash command script that sets up a server on a local machine that hosts a website. By simply running this file, it sets up ngninx configurations and creates an html file. This demonstrates a computer program interacting with the operating system and setting up a network configuration.


  • To understand the concepts and techniques of software design.

    Mathemon is a project created for the Software Engineering class. This is a group effort and demonstrates the designing and writing of software. The project was built with Javascript and used a Firebase database.


  • To acquire significant project experience working both individually and in a group setting.

    Chat Server was a pair programming project built with Andrew Luo in Networks class. This project involved learning node.js and builidng it collabratively. This project especially stands out as a group effort the software requires at least two users.


  • To develop effective problem solving skills

    The work required for the Utah Animal Adoption Center application was extensive. No project at Westminster has required the same degree of learning information on your own. The concept of a front end and back end were completely new to me, so completing this project took a lot of patient learning of new skills.
